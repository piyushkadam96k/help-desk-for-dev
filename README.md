## ✨ Key Features

- **🚀 Lightning Fast & Offline**: Developed entirely with vanilla HTML, CSS, and JS (No heavy frameworks like React or Vue). Works perfectly without an internet connection.
- **🛡️ Manifest V3 Compliant**: Uses modern browser APIs like `declarativeNetRequest` for strict adherence to Chrome's latest security and privacy standards.
- **🎨 Premium UI/UX**: Features a beautiful dark-mode interface with subtle glassmorphism, fluid typography, and dynamic animations.
- **🚫 Built-in Site Blocker**: Distracted? Turn on **Focus Mode** and block distracting websites dynamically, straight from your developer toolkit.

## 💻 Installation (Developer Mode)

Since this extension isn't publicly listed in the Chrome Web Store yet, you can easily install it locally in a few seconds:

1. **Download the code**: Clone or download this repository to your local machine.
2. **Open Extensions Page**: Open Google Chrome and navigate to `chrome://extensions/`
3. **Enable Developer Mode**: Toggle the **Developer mode** switch in the top-right corner.
4. **Load the Extension**: Click the **Load unpacked** button in the top-left corner.
5. **Select the Folder**: Select the folder containing `manifest.json`.
6. **Pin it!**: Click the puzzle piece icon 🧩 next to your address bar and pin the **DevForge** extension for easy access!

---

## 🔒 Privacy First

DevForge doesn't track you. 
- All data formatting, generation, and decoding happens **100% locally** in your browser. 
- The Focus Mode website blocker only requests permissions to intercept network traffic (`declarativeNetRequest`) for the sole purpose of blocking the domains you manually specify. No browsing history is tracked or transmitted.
